# Pembe Flour Millers 🌾

A fully responsive e-commerce website for Pembe Flour Millers — a Kenyan flour and animal feed brand based in Nairobi. Built from scratch using HTML, CSS and JavaScript.

---

## 🔗 Live Site
👉 [Click here to visit the live website](https://mulei-tech27.github.io/pembe-flour-millers)

---

## 📸 Pages
| Page | Description |
|------|-------------|
| **Home** | Hero section, features, product preview, stats, testimonials |
| **Products** | Full product catalogue with shopping cart |
| **About** | Company story, mission and values |
| **Contact** | Contact form and WhatsApp direct button |

---

## ✨ Features
- 🛒 Shopping cart with add/remove and quantity controls
- 🔍 Product search and filter by category
- 💬 WhatsApp and email ordering system
- 📱 Fully responsive with mobile hamburger menu
- 🌟 Splash/loading screen with animation
- 🎬 Scroll fade-in animations
- ⭐ Customer testimonials section
- 🔝 Back to top button
- 💬 Floating WhatsApp button on all pages
- 🚫 Custom 404 error page
- ⚡ Page transition animations
